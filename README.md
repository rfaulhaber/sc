# sc

A terminal-based stack calculator, written in Rust.

## Installing

Currently you must manually build this project using `cargo`.
You can do this by running:

```
cargo install --git https://github.com/rfaulhaber/sc
```

## Use

While the program is running, enter values to add them onto the stack. When
you want to perform an operation, enter any valid operator.

For example:

```
1
2
3
+
p
5
+ 6 /
p
1
```
